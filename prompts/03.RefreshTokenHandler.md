Context:
- controller/auth.controller.ts: refreshHandler, logoutHandler
- lib/token.ts: verifyRefreshToken (uses process.env.JWT_ACCESS_SECRET)
- routes/auth.router.ts: POST /refresh -> refreshHandler, POST /logout -> logoutHandler

refreshHandler:
1. Extract refreshToken from cookies.
2. Decode payload via verifyRefreshToken(token).
3. Query DB for user by payload.sub.
4. Abort if user.tokenVersion !== payload.tokenVersion.
5. Generate new accessToken and refreshToken (include id, role, tokenVersion).
6. Set 'refreshToken' cookie: httpOnly=true, secure=(NODE_ENV==='production'), sameSite='lax', maxAge=7d.
7. Return 200 with refreshed accessToken.

logoutHandler:
1. Clear 'refreshToken' cookie (path: "/").
