Context:
- controller/auth.controller.ts: forgotPasswordHandler, resetPasswordHandler
- routes/auth.router.ts:
    - POST /forgot-password -> forgotPasswordHandler
    - POST /reset-password -> resetPasswordHandler

forgotPasswordHandler:
1. Extract and normalize email (lowercase, trimmed) from req.body.
2. Query DB for user by email. If not found, return 200 (generic success message for security).
3. Generate rawToken = crypto.randomBytes(32).toString('hex').
4. Generate tokenHash = crypto.createHash('sha256').update(rawToken).digest('hex').
5. Set user.resetPasswordToken = tokenHash and user.resetPasswordExpires = Date.now() + 15 mins.
6. Save user changes to DB.
7. Construct resetUrl = `${process.env.APP_URL}/reset-password?token=${rawToken}`.
8. Use sendMail to deliver resetUrl to user's email.
9. Return 200 with generic success message.

resetPasswordHandler:
1. Extract rawToken and new password from request.
2. Generate tokenHash = crypto.createHash('sha256').update(rawToken).digest('hex').
3. Query DB for user where resetPasswordToken === tokenHash AND resetPasswordExpires > Date.now().
4. If user not found or expired, return 400 "Invalid or expired token".
5. Hash new password using hashPassword.
6. Update user fields:
   - passwordHash = newPasswordHash
   - resetPasswordToken = undefined
   - resetPasswordExpires = undefined
   - Increment tokenVersion by 1 (invalidates active sessions)
7. Save user changes to DB.
8. Return 200 with success message.
