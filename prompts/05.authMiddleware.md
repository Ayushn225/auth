middleware/requireAuth.ts:
1. Extract 'Authorization' header from req.
2. Abort 401 if header missing or doesn't start with "Bearer ".
3. Extract token string (split by " ").
4. Decode payload via verifyAccessToken(token). Abort 401 if invalid/expired.
5. Query DB for user by payload.sub. Abort 401 if not found.
6. Abort 403 if user.tokenVersion !== payload.tokenVersion.
7. Extend Express Request type (or typecast `req as any`) and attach:
   req.user = { 
       userId: user._id, 
       userEmail: user.email, 
       userName: user.name, 
       role: payload.role, 
       userEmailVerified: user.isEmailVerified 
   }
8. Call next(). Export requireAuth.

routes/user.routes.ts:
1. Initialize Express Router.
2. Define GET /me protected by requireAuth middleware.
3. Handler logic: Extract req.user and return res.json({ user: req.user }).
4. Export router.

app.ts:
1. Import userRouter.
2. Register middleware: app.use("/api/users", userRouter).
