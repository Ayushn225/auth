middleware/requireRole.ts:
1. Create a factory function requireRole(allowedRoles: ('user' | 'admin')[]) that returns an Express middleware: (req, res, next) => void.
2. Inside the middleware, check if req.user exists and if allowedRoles.includes(req.user.role).
3. If true, call next().
4. If false or req.user is missing, abort with 403 "Forbidden: Insufficient permissions".
5. Export requireRole.

routes/admin.routes.ts:
1. Initialize Express Router.
2. Define GET /users protected by [requireAuth, requireRole(['admin'])].
3. Handler logic:
   - Query DB for all users, sorted by createdAt in descending order.
   - Return 200 with the users array in the JSON response.
   - Wrap in try/catch; return 500 on internal errors.
4. Export router.

app.ts:
1. Import adminRouter.
2. Register middleware: app.use("/api/admin", adminRouter).
